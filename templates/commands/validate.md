# /validate - 创意验证系统 🎯

> **核心功能**: 通过 Twitter/YouTube 数据分析验证创意潜力
> **支持**: 手动输入、Twitter 分析、YouTube 分析

---


脚本返回:

## 第五步: 更新创意状态 (自动)

当用户开始研究或创作时,状态会自动更新:

```bash
# AI 操作: 更新状态 (在 /research 或 /write 命令中自动调用)
bash scripts/bash/validate.sh update-status [topic_id] [new_status]
```

状态流转:
- `pending` → `researching` (开始研究时)
- `researching` → `writing` (开始创作时)
- `writing` → `distributed` (分发完成时)

## 📊 创意评分系统 (未来功能)

当 Twitter/YouTube API 集成后,将自动计算创意评分:

**评分维度**:
- 互动数据 (40%): 点赞、转发、评论
- 传播速度 (30%): 增长趋势
- 受众质量 (20%): 评论深度、情感
- 话题热度 (10%): 相关话题的整体热度

**评分范围**: 0-100
- 90-100: 极高潜力 🔥
- 70-89: 高潜力 ⭐
- 50-69: 中等潜力 ✓
- 30-49: 较低潜力 ⚠️
- 0-29: 低潜力 ❌

## 💡 使用技巧

1. **先手动输入,后 API 验证**
   - 快速建立创意库
   - 后续可通过 API 补充数据

2. **批量验证**
   - 一次输入多个创意
   - 统一进行研究和筛选

3. **定期更新**
   - 每周添加新创意
   - 保持创意库的新鲜度

4. **结合研究**
   - 验证后立即进行研究
   - 建立完整的知识体系
